### Day 1 — February 2026
*Topic :* Linux file system structure  
*Commands practiced :*
- ls -la — list files with permissions
- cd /etc — navigate to config folder
- chmod 755 file — change permissions
- cat /etc/passwd — read system users file

*What I understood :* The Linux file system starts at / (root). Every file and folder is under it.  
*What was difficult :* Understanding numeric permissions (755, 644, 600)
